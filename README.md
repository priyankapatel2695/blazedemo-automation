# BlazeDemo Automation Framework

## Project Description
This project automates the BlazeDemo flight booking application using Selenium WebDriver, Java, TestNG, Maven, and the Page Object Model (POM).

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Git
- GitHub

## Project Structure
- src/main/java
- src/test/java
- pom.xml
- testng.xml

## How to Run
1. Open the project in Eclipse.
2. Update Maven dependencies.
3. Run testng.xml as TestNG Suite.
